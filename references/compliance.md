# 内容合规审核（日报输出前必做）

日报会**上公网站 + 推送到内部 IM（如京ME）**、给团队同事看——所以最终**呈现 / 落盘 / 分发之前**，必须过一轮合规审核。这是「输出前」最后一道闸：**合规优先于完整，宁可少一条，不留一个风险**。

AI 情报日报天然涉及大量**中美 AI 竞争、出口管制、监管、制裁、地缘**话题，客观中立是底线——这一道闸不能省。

## 审核清单（逐条过，命中即处理）

| # | 检查项 | 处理方式 |
|---|---|---|
| 1 | **政治 / 地缘表述中立** —— 涉及中美竞争、出口管制、监管、制裁、地缘的条目，是否**只客观陈述事实**（谁做了什么、何来源），未加评判性立场、未渲染对立情绪、未传播敏感政治观点 | 改写成中立事实陈述（「X 宣布 / 据 Y 报道…」），删去立场性形容词与情绪渲染；纯政治敏感、与 AI 产业关系弱的，降级或移除 |
| 2 | **信息真实、不传谣** —— 未证实传闻是否已按信源置信度护栏降级（不进必读、标「来源存疑」） | 未证实的移出必读、标注存疑；查无一手来源的移除 |
| 3 | **导向正面** —— 是否渲染 AI 威胁论 / 大规模失业恐慌 / 极端论调 | 负面事件**客观陈述**即可，不煽动、不放大恐慌 |
| 4 | **来源合规** —— 每条是否标注来源；有无引用不可信 / 违规源 | 补来源；移除不可信源内容 |
| 5 | **企业表述谨慎** —— 涉及具体公司（尤其国内友商 / 竞品）的负面，是否用**事实 + 来源**而非主观贬低 | 改为「据 X 报道」式事实陈述，去掉主观评价词 |
| 6 | **隐私与敏感** —— 人物动态（X 等）是否只用**公开信息**，不涉个人隐私 / 未公开内容 | 移除涉私 / 未公开内容 |
| 7 | **无违规内容** —— 无色情 / 暴力 / 歧视 / 违法导向 | 直接移除 |

## 执行

1. 日报生成后、**呈现 / 落盘 / 分发之前**，逐条过上表（尤其第 1 条政治地缘中立，AI 情报里最高频）。
2. 命中的条目按「处理方式」**改写中立化**或**降级 / 移除**——宁可少一条，不留一个合规风险。
3. 全部通过后，可在尾注标「· 已过合规审核」，让分发方安心。

> **原则**：① **合规优先于完整**——拿不准的条目，降级或省略，别赌；② 中立 ≠ 删除事实——重要事实照报，只是**去立场、去情绪、标来源**；③ 这道闸对所有 lens、所有输出渠道（网页 / IM / 落盘）一视同仁。
